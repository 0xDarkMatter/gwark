description: View the project roadmap and development status
---

Read and display the project roadmap from docs/ROADMAP.md. Show the current phase, development status, and the full roadmap in a clear, formatted way suitable for terminal display.

For the current phase section, format task lists with checkboxes:
- Completed items: [x]
- In Progress items: [-]
- Remaining items: [ ]

Example format:
```
✅ Completed
- [x] Project scaffolding and structure
- [x] OAuth2 authentication flow

🔨 In Progress
- [-] MCP server implementation
- [-] Complete MCP tool registration

📋 Remaining
- [ ] Configuration management finalization
- [ ] Error handling improvements
```
